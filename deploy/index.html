Пошаговая схема интеграции

Этап 1: Calltouch → Б24 2-3 часа

  1. ЛК Calltouch → Интеграции → Битрикс24
  2. Авторизация OAuth (домен Б24)
  3. Настройка: звонок → создать лид
  4. Маппинг полей (телефон, UTM, источник)
  5. Обратная синхронизация (статус сделки → CT)
  6. Тест: позвонить на подменный номер

Этап 2: Директ → Calltouch 30 мин

  1. ЛК Calltouch → Интеграции → Яндекс.Директ
  2. OAuth авторизация в Яндексе
  3. Выбор кампаний для отслеживания
  4. Автоимпорт: расходы, показы, клики, CTR

Этап 3: UTM-метки в Директе 30 мин

?utm_source=yandex
&utm_medium=cpc
&utm_campaign={campaign_name}
&utm_content={ad_id}
&utm_term={keyword}

Этап 4: Сквозная в Метрике Бесплатно

  1. Метрика → Настройка → Сквозная аналитика
  2. Расходы Директа подтянутся авто
  3. Передать ClientID Метрики в CRM через формы
  4. Выгрузить закрытые сделки через API

SWOT-анализ Espace Agency

Сильные стороны (Strengths)

  • 50+ кейсов в 30+ нишах — широкая экспертиза
  • Перформанс-фокус с измеримым ROI (ROMI 500%+)
  • Компактная команда — гибкость и скорость
  • Мультиканальность: таргет, контекст, SEO, блогеры
  • Прозрачная отчётность (еженедельно + ежемесячно)
  • Основатель = лицо бренда (личный бренд Ангелины)

Слабые стороны (Weaknesses)

  • ИП, не ООО — ограничения для крупных тендеров
  • Нет публичной финансовой отчётности (ИП)
  • Масштабируемость ограничена размером команды
  • Нет рейтингового присутствия (Tagline, Ruward, AdIndex)
  • Отсутствие Retail Media / маркетплейс-экспертизы
  • Два домена (espace-agency.ru + агентсво.рф) — SEO-каннибализация

Возможности (Opportunities)

  • AI-инструменты снижают расходы и масштабируют услуги
  • Telegram Ads — растущий канал (+300%/год)
  • Retail Media — новый сегмент с огромным потенциалом
  • Сквозная аналитика — можно предлагать как доп. услугу
  • AI-сайты — быстрое создание лендингов для клиентов
  • 10-15% мелких агентств уходят — их клиенты ищут замену

Угрозы (Threats)

  • Медиаинфляция +20-40% — рост CPL для клиентов
  • Крупные агентства поглощают рынок (ТОП-5 = 40%+ выручки)
  • AI-автоматизация Яндекса снижает ценность ручного управления
  • Кадровый дефицит — зарплаты +15-25%/год
  • Клиенты уходят в in-house с AI-инструментами
  • Регуляторное давление (маркировка, ОРД)
